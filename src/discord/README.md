# طبقة Discord (محولات)

هذا المجلد مخصص لدمج خدمات النواة مع discord.js دون منطق أعمال مباشر. الهدف في المراحل القادمة:

- تعريف معالجات للأوامر والأحداث تستدعي خدمات `src/core` عبر `createCoreContext`.
- الحفاظ على كل نصوص الردود باللغة العربية مع بقاء المعرفات/slugs بالإنجليزية.
- عزل متغيرات التهيئة (القنوات، الخيوط، المعرفات) هنا حتى لا تتسرب إلى النواة.

> ملاحظة: لا يوجد تنفيذ بعد. `src/index.js` لا يزال يعرّف الأحداث مباشرة، وسيتم نقله تدريجياً إلى هذه الطبقة.
